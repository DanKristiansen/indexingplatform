    Makes a free-text search in all indexed elements and returns a list of LightXmlObject elements with the results

    • Parameters:
      • $studyId - the ID of the StudyUnit that needs to be matched
      • $title - the Title of the Study that needs to be matched
      • $abstract-purpose - the Abstract or Purpose that needs to be matched
      • $creator - the Creator that needs to be matched
      • $kindOfData - the KindOfData that needs to be matched
      • $coverageFrom - start-date of the temporal range
      • $coverageTo - end-date of the temporal range
      • $concept - the Concept that needs to be matched
      • $universe - the Universe that needs to be matched
      • $question - the Question that needs to be matched
      • $variable - the Variable that needs to be matched
      • $category - the Category that needs to be matched
      • $hits-perpage - number of hits to be shown per page
      • $hit-start - number of the first hit to be shown on the page
